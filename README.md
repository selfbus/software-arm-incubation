# software-arm-incubation

This repository contains Selfbus software projects for the ARM microcontrollers (e.g. NXP LPC1115).

## Contributing

- Contributions are welcome!
- Please submit a pull request or open an issue for any bugs or feature requests.
- Please try to follow the existing code style and conventions when contributing.
- An MCUxpresso code style formatter can be found [here](mcuxpresso-code-style-formatter.xml).
  It can be imported into the MCUxpresso IDE via:  
  `Window -> Preferences -> C/C++ -> Code Style -> Formatter -> Import...`.
